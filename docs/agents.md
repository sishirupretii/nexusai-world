# Agent Documentation

## Core Agents

### ArchitectAgent
- Designs world structures
- Plans district expansions
- Merges approved PRs

### BuilderAgent
- Generates code changes
- Creates pull requests
- Manages deployments

### ResearchAgent
- Analyzes data patterns
- Benchmarks performance
- Proposes improvements

### SecurityAgent
- Monitors threats
- Reviews code changes
- Blocks malicious actions

### EvolutionAgent
- Improves agent strategies
- Cross-pollinates patterns
- Drives system evolution
